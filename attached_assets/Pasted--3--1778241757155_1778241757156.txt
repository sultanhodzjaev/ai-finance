## КОНТЕКСТ

В существующем проекте уже работает:
- Telegram-бот на aiogram 3.x с Gemini API — записывает **расходы** через текст и фото чеков
- AI-финансист в боте отвечает на вопросы о тратах
- Telegram Mini App с дашбордом, историей, добавлением и редактированием транзакций
- FastAPI отдаёт REST API для Mini App
- Хранение в Python-словаре в памяти (`services/storage.py`)

**Твоя задача:** добавить полноценный учёт **доходов** — параллельно с существующими расходами, не ломая то что уже работает.

**ВАЖНО:** не пересоздавай проект, не переписывай существующую логику расходов. Только дополняй.

### Эндпоинт `/api/stats` (GET)

Полностью обновляем — теперь возвращает доходы, расходы и баланс:

```python
@router.get("/stats")
async def get_stats(x_init_data: str = Header(...)):
    telegram_id = require_auth(x_init_data)
    txs = get_transactions(telegram_id)
    now = datetime.now()
    month_txs = [
        tx for tx in txs
        if datetime.fromisoformat(tx["datetime"]).year == now.year
        and datetime.fromisoformat(tx["datetime"]).month == now.month
    ]

    expenses = [tx for tx in month_txs if tx.get("type", "expense") == "expense"]
    incomes = [tx for tx in month_txs if tx.get("type") == "income"]

    expense_by_category = {}
    for tx in expenses:
        cat = tx["category"]
        expense_by_category[cat] = expense_by_category.get(cat, 0) + tx["amount"]

    income_by_category = {}
    for tx in incomes:
        cat = tx["category"]
        income_by_category[cat] = income_by_category.get(cat, 0) + tx["amount"]

    total_expense = sum(expense_by_category.values())
    total_income = sum(income_by_category.values())

    # По дням — две серии
    expense_by_day = {}
    income_by_day = {}
    for tx in expenses:
        day = datetime.fromisoformat(tx["datetime"]).strftime("%Y-%m-%d")
        expense_by_day[day] = expense_by_day.get(day, 0) + tx["amount"]
    for tx in incomes:
        day = datetime.fromisoformat(tx["datetime"]).strftime("%Y-%m-%d")
        income_by_day[day] = income_by_day.get(day, 0) + tx["amount"]

    return {
        "total_income": total_income,
        "total_expense": total_expense,
        "balance": total_income - total_expense,
        "transaction_count": len(month_txs),
        "expense_by_category": expense_by_category,
        "income_by_category": income_by_category,
        "expense_by_day": expense_by_day,
        "income_by_day": income_by_day
    }
```

### Эндпоинт `/api/transactions` (GET) — фильтрация

Добавь опциональный query-параметр `type` для фильтрации:

```python
@router.get("/transactions")
async def list_transactions(
    type: str | None = None,  # НОВОЕ: "income", "expense" или None (все)
    x_init_data: str = Header(...)
):
    telegram_id = require_auth(x_init_data)
    txs = get_transactions(telegram_id)
    if type in ["income", "expense"]:
        txs = [tx for tx in txs if tx.get("type", "expense") == type]
    txs_sorted = sorted(txs, key=lambda x: x["datetime"], reverse=True)
    return {"transactions": txs_sorted}
```
